# DUAL.AI.TEACHer Website

Public landing site for the DUAL.AI.TEACHer project, built with Astro and Tailwind CSS.

## Tech Stack

- Astro
- Tailwind CSS (via `@tailwindcss/vite`)
- Static deployment to GitHub Pages

## Local Development

```sh
npm install
npm run dev
```

Open `http://localhost:4321`.

## Build

```sh
npm run build
npm run preview
```
